# ShootingGame
# Android studio를 이용한 횡방향 슈팅게임이다.

Game Framework를 개발후 비행기 이미지와 Enemy 이미지, 애니메이션 이미지를 적용해 개발하였다.

## 2탄 보스
<img width="1149" alt="스크린샷 2020-12-17 16 35 49" src="https://user-images.githubusercontent.com/67869514/102457679-79e86200-4086-11eb-8509-5eb5eff0a73c.png">


3탄탄까지 있고 보스는 체력과 공격속도가 빠르다.
Z키를 누르면 큰 미사일이 발사되며
스페이스바를 누르면 일반 미사일이 발사된다
박쥐는 HP가 10
초록색 용은 HP가 30
Boss 체력은 탄마다 다르다.
피격시 1초정도 무적상태가 된다.
